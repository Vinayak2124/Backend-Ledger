📘 Ledger Backend API

Backend service for the Ledger / Transaction Management System.
Handles authentication, accounts, transactions, and balance tracking.

🚀 Features

✔ User Authentication (JWT-based)
✔ Account Management
✔ Ledger / Transaction Recording
✔ Balance Tracking
✔ Secure Cookie-based Sessions
✔ MongoDB Transactions Support
✔ Error Handling & Validation

🛠 Tech Stack

Node.js

Express.js

MongoDB / Mongoose
